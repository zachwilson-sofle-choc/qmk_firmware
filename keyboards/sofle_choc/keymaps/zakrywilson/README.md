## Compiling

```sh
qmk compile -kb sofle_choc -km zakrywilson
```

## Flashing

```sh
qmk flash -kb sofle_choc -km zakrywilson
```

## Generating ascii keymap

### Mac

```sh
rg "\* " --glob keymap.c | cut -c 13- | pbcopy
```

### Linux

```sh
xclip -sel clip $(rg "\* " --glob keymap.c | cut -c 12-)
```
