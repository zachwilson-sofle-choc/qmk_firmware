## Compiling

```sh
qmk compile -kb sofle_choc -km arnjwlsn
```

## Flashing

```sh
qmk flash -kb sofle_choc -km arnjwlsn
```

## Generating ascii keymap

### Mac

```sh
rg "\* " --glob keymap.c | cut -c 13- | pbcopy
```

### Linux

```sh
rg "\* " --glob keymap.c | cut -c 12- | xclip -sel clip
```
